# Stylowe Wnętrza

Statyczna strona dla salonu podłóg i drzwi Stylowe Wnętrza w Pabianicach.

## Co zawiera

- polska treść zoptymalizowana pod lokalne SEO,
- widoczny telefon i adres e-mail bez JavaScript,
- sekcję opinii Google,
- dane strukturalne LocalBusiness i FAQ,
- formularz kontaktowy mailto,
- mapę Google,
- `robots.txt` i `sitemap.xml`.

## Uruchomienie lokalne

Otwórz `index.html` w przeglądarce albo uruchom prosty serwer statyczny:

```bash
python3 -m http.server 8000
```
